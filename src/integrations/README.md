# OpenAI Integration (Phase 3)

Example code for integrating OpenAI function calling with Precogs.

## Dependencies

This integration requires the OpenAI SDK:

```bash
npm install openai
```

## Environment Variables

```bash
OPENAI_API_KEY=sk-...
```

## Files

- `openai-example.js` - Example implementation showing:
  - Function calling with streaming
  - Function calling without streaming (simpler)
  - HTTP endpoint example

## Usage

See `FUNCTION_CALLING_INTEGRATION.md` for complete guide.
